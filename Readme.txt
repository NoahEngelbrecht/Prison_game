Prison Game - Readme.txt av Noah Engelbrecht (2025-05-13

  Översikt
Prison Game är ett textbaserat äventyrsspel där spelarna antar rollen som en fånge som försöker fly från ett högsäkerhetsfängelse. Spelet kräver att spelarna löser pussel, fattar beslut och interagerar med olika karaktärer för att framgångsrikt navigera genom fängelset och slutligen uppnå frihet.

  Funktioner
Interaktivt Gameplay: Spelare kan välja olika vägar och fatta beslut som påverkar spelets utfall.
Flera Rum och Celler: Spelet har flera rum och celler att utforska, var och en med unika utmaningar och interaktioner.
Strid och Övertalning: Spelare kan välja att slåss mot vakter eller försöka övertala dem att låta dem passera.
Dolda Föremål: Det finns en chans att upptäcka dolda föremål som kan hjälpa till i flykten.
Prestationer: Spelare kan låsa upp prestationer baserat på sina handlingar under spelets gång.
Komma igång
För att spela spelet:


  Spelinstruktioner
När spelet startar kommer spelarna att presenteras med en huvudmeny med följande alternativ:

Spela spelet: Börja ditt äventyr i fängelset.
Läsa instruktionerna: Visa denna instruktion igen.
Avsluta spelet: Stäng spelet.

  Spelmekanik

Utforskning: Spelarna börjar i sin cell och kan välja att kolla runt i rummet eller öppna dörren för att gå ut.
Val: Under spelets gång kommer spelarna att fatta beslut som påverkar berättelsen. Till exempel kan de välja att gå vänster eller höger i korridoren eller välja vilket rum de vill gå in i.
Interagera med vakter: I vissa rum kommer spelarna att stöta på vakter. De kan välja att slåss eller försöka övertala vakterna att låta dem passera.
Hitta Nycklar och Föremål: Spelare kan hitta nycklar som hjälper dem att låsa upp dörrar och andra dolda föremål som kan vara till hjälp i deras flykt.
Prestationer: Spelare kan låsa upp prestationer genom att utföra specifika uppgifter, som att hitta nycklar eller dolda föremål.

  Exempel på Handlingar

När du uppmanas att "kolla runt" eller "öppna dörren", skriv ditt val och tryck på Enter.
Om du går in i ett rum med en vakt kan du välja att "slåss" eller "försöka övertala".
Om du hittar ett föremål får du en notifikation och kan ha möjlighet att använda det i framtida möten.

  Tips för Framgång

Utforska noggrant: Kolla varje rum och cell eftersom det kan finnas dolda föremål som kan hjälpa dig att fly.
Var uppmärksam på vakterna: Dina interaktioner med dem kan leda till olika utfall.
Håll koll på dina prestationer: De kan ge insikter om din framsteg och framgång inom spelet.
Avslutning
Prison Game erbjuder en spännande och utmanande upplevelse där dina beslut och handlingar avgör om du lyckas fly eller inte. Lycka till, och må du finna friheten!
